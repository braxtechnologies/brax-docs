## ğŸ’° How Management Compensation Is Calculated

<p style="font-size: 1.05rem; color: #555; margin-bottom: 1.5rem;">
  Management roles use the same base formula as ICs, with an added Long-Term Incentive component:
</p>

```text path=null start=null
Base Compensation = max(LMV, Adjusted IMV)
where Adjusted IMV = IMV Ã— (1 - tax_rate) Ã— COL_multiplier

Total Compensation = Base + LTI
where LTI = (team performance % + company goals %) Ã— base Ã— level_multiplier
```

<div style="background: #f8f9fa; padding: 1rem; border-radius: 6px; margin: 1rem 0 2rem 0;">
  <strong>Key Terms:</strong>
  <ul style="margin: .5rem 0 0 1rem; line-height: 1.7;">
    <li><strong>LMV</strong> (Local Market Value): Highest trusted local benchmark + 10% buffer</li>
    <li><strong>IMV</strong> (International Market Value): Global benchmark adjusted for tax and COL</li>
    <li><strong>LTI</strong> (Long-Term Incentive): Variable comp tied to team/company performance</li>
    <li><strong>Level multiplier</strong>: Ranges from 0.10 (Level 1) to 0.40 (Level 4)</li>
  </ul>
</div>
