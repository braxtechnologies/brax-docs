# Murena (/e/OS)

Murena (formerly /e/OS) is a privacy-focused mobile operating system based on Android that removes all Google services and provides its own privacy-respecting ecosystem.

## Features

- Complete Google-free experience
- Privacy score for apps
- Integrated cloud services
- Advanced Privacy features
- App Lounge for app installation
- BlissLauncher interface
