# CalyxOS

CalyxOS is a privacy-focused Android mobile operating system that provides a Google-free environment with strong privacy and security features.

## Features

- Privacy by design
- microG for app compatibility
- Datura firewall for network control
- Encrypted backups
- Verified boot
- Regular monthly security updates
