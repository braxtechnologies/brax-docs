# BraX3 Overview

## Device Information

The BraX3 is Brax Technologies' flagship secure smartphone, designed for privacy-conscious users who demand both security and functionality.

## Supported Operating Systems

The BraX3 supports a variety of custom Android-based operating systems and alternative mobile platforms:

- **AOSP** - Android Open Source Project
- **BraxOS** - Brax's custom secure operating system
- **CypherOS** - Privacy-focused Android distribution
- **iod√©OS** - Privacy-focused Android OS
- **LineageOS** - Popular open-source Android distribution
- **LunarOS** - Custom Android-based operating system
- **Ubuntu Touch** - Ubuntu mobile operating system

## Getting Started

Before installing any custom operating system on your BraX3, you'll need to:

1. [Unlock the bootloader](unlock-bootloader.md)
2. Install Android Platform Tools (ADB/Fastboot)
3. Install the appropriate USB drivers for your computer

## Device Specifications

- **Processor**: MediaTek chipset
- **Bootloader**: Unlockable
- **Fastboot Mode**: Supported
