# Brax Technologies

## Overview

Brax Technologies is a company focused on secure and private mobile products. The company develops privacy-first technology solutions designed to keep user data safe and give users control over their personal information.

## Mission

Brax Technologies is building a new path for mobile technology, centered on privacy, security, and user empowerment. The company prioritizes hardened, open-source software, guaranteed security updates, and advanced privacy controls.

## Products and Services

### BraX3

The flagship product, BraX3, is a privacy-focused smartphone designed from the ground up with security in mind. Key features include:

- Hardened, open-source software
- Guaranteed security updates
- Advanced privacy controls
- User data protection and control

**Crowdfunding Success**: BraX3 is one of only 4 smartphone crowdfunding campaigns to exceed $1M in funding, demonstrating strong market demand for privacy-focused mobile devices.

### Brax Connect

Brax Technologies has launched Brax Connect, described as "the most secure mobile service," providing secure connectivity solutions for privacy-conscious users.

## Partnerships

Brax Technologies partners with **UBports** to support the development of Ubuntu Touch mobile OS, furthering the company's commitment to open-source mobile operating systems.

## Affiliate Program

The company operates an affiliate program that allows participants to earn while promoting privacy-focused, fairly-priced technology.
