## ğŸ’° How IC Compensation Is Calculated

<p style="font-size: 1.05rem; color: #555; margin-bottom: 1.5rem;">
  All IC roles use the same transparent formula, adjusted for local market and cost of living:
</p>

```text path=null start=null
Compensation = max(LMV, Adjusted IMV)
where Adjusted IMV = IMV Ã— (1 - tax_rate) Ã— COL_multiplier
```

<div style="background: #f8f9fa; padding: 1rem; border-radius: 6px; margin: 1rem 0 2rem 0;">
  <strong>Key Terms:</strong>
  <ul style="margin: .5rem 0 0 1rem; line-height: 1.7;">
    <li><strong>LMV</strong> (Local Market Value): Highest trusted local benchmark + 10% buffer</li>
    <li><strong>IMV</strong> (International Market Value): Global benchmark (PayScale, Glassdoor, etc.)</li>
    <li><strong>Tax rate</strong>: Standard deduction (e.g., 38% for U.S. benchmarks)</li>
    <li><strong>COL multiplier</strong>: Cost-of-living adjustment (e.g., 0.75 for Sofia, Bulgaria)</li>
  </ul>
