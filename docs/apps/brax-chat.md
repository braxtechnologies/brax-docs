# Brax Chat

## Overview

Brax Chat is a privacy-focused, secure messaging application designed for the Brax ecosystem. Built with end-to-end encryption and decentralized architecture, it ensures your conversations remain private and secure.

## Key Features

- **End-to-End Encryption**: All messages are encrypted by default
- **Decentralized Architecture**: No central server storing your data
- **Cross-Platform**: Available on Android, iOS, and desktop
- **Open Source**: Fully transparent and auditable code
- **Group Messaging**: Secure group chats with up to 500 members
- **File Sharing**: Share encrypted files up to 100MB
- **Voice & Video Calls**: Encrypted audio and video communication

## Technical Stack

- **Frontend**: React Native for mobile, Electron for desktop
- **Encryption**: Signal Protocol
- **Backend**: Distributed nodes with Matrix protocol
- **Storage**: Local encrypted database

```bash

cd brax-chat
npm install
npm run build
```
